# Sprint 001: Project Setup

**Objective:** Establish the foundational file structure for a simple, static portfolio using vanilla web technologies.

---

### What Changed

- **`index.html` created**: A single HTML file was created to serve as the main page for all content. It includes the basic document structure and links to the CSS and JavaScript files.
- **`style.css` created**: A single CSS file was created to contain all styling for the portfolio.
- **`script.js` created**: A single JavaScript file was created to handle any dynamic functionality, such as rendering projects.

---

### Acceptance Criteria

- [x] A single `index.html` file is created.
- [x] A single `style.css` file is created and linked in the HTML.
- [x] A single `script.js` file is created and linked in the HTML.
